# OnlyToType

タイピングで操作する2Dアクションゲーム（Unity）。

## 概要

自動スクロールするステージを、画面に表示される文字をタイピングして進む。敵を倒し、障害物を回避しながらゴールを目指す。

## Features

- **Tutorial system** — jump/dash の操作をガイド。ウェイポイントヒント、ライフ (3)、死亡/リスポーン、GameOver パネル、ゴール旗で完了。初回は強制チュートリアル。`skip` とタイプしてスキップ可能。
- **Random Stage** — 自動生成の横スクロールステージ。
- **Title menu** — `1`=Tutorial, `2`=Random Stage

## 操作方法

| 操作 | キー |
|------|------|
| ジャンプ | `jump` / `hop` と入力 |
| ダッシュ | `dash` / `run` と入力 |
| 決定 / 再スタート | Enter |
| タイトルに戻る | Space（ゲームオーバー/クリア時）
