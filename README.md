# OnlyToType

タイピングで操作する2Dアクションゲーム（Unity）。

## 概要

自動スクロールするステージを、画面に表示される文字をタイピングして進む。敵を倒し、障害物を回避しながらゴールを目指す。

## 操作方法

| 操作 | キー |
|------|------|
| ジャンプ | 画面上の文字をタイピング |
| 決定 / 再スタート | Enter |
| タイトルに戻る | Space（ゲームオーバー/クリア時） |
